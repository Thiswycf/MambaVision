# MambaVision 项目学习与运行指南（中文）

## 项目用途

`MambaVision` 是一个视觉骨干网络项目（Hybrid Mamba + Transformer），主线是图像分类，并扩展到目标检测与语义分割。

## 关键目录

- `mambavision/`：分类模型、训练与验证脚本
- `object_detection/`：基于 MMDetection 的检测代码
- `semantic_segmentation/`：基于 MMSegmentation 的分割代码
- `requirements.txt`：Python 依赖入口
- `setup.py`：包元信息，声明 `python>=3.9`

## 环境与版本

- 推荐 conda 环境名：`mambavision`
- 推荐 Python：`3.10`

```bash
conda create -n mambavision python=3.10 -y
conda activate mambavision
```

## 依赖安装（稳妥顺序）

先装与本机 CUDA 匹配的 PyTorch，再安装项目依赖，可降低 `mamba-ssm` 安装失败概率。

```bash
pip install torch==2.6.0 torchvision==0.21.0 --index-url https://download.pytorch.org/whl/cu124
pip install -r requirements.txt
```

若 `mamba-ssm==2.2.4` 失败，再尝试：

```bash
pip install mamba-ssm==2.2.4 --no-build-isolation
```

## 最小可运行验证

### 1) 依赖导入检查

```bash
python -c "import torch; print(torch.__version__, torch.version.cuda)"
```

### 2) 项目导入检查

```bash
python -c "import mambavision; from mambavision.models.mamba_vision import create_model; print('import ok')"
```

### 3) 分类最小前向（无数据集）

```bash
python - <<'PY'
import torch
from mambavision import create_model

device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
model = create_model('mamba_vision_T2').to(device).eval()

x = torch.randn(1, 3, 224, 224).to(device)
with torch.no_grad():
    y = model(x)
print('forward ok:', tuple(y.shape))  # 输出: (1, 1000)
PY
```

## 常见失败排查

1. `mamba-ssm` 安装失败：优先检查 CUDA 与 `torch.version.cuda` 是否一致。
2. `ModuleNotFoundError`：确认已激活 `mambavision` 环境且安装在该环境内。
3. 数据路径报错：分类通常需要 `train/val` 或 `train/validation` 结构；检测/分割按各自 README 准备。
4. 显存不足：先减小 `batch-size`，优先跑导入检查与单步前向。
